5.尋找哪個經銷商平均囤貨時間最久

Find those dealers who keep a vehicle in inventory for the longest average time.

SELECT DATEDIFF(transaction.date, vehicles.transaction_date) AS DiffDate,transaction.dealer FROM transaction INNER JOIN vehicles ON transaction.VIN=vehicles.VIN GROUP BY transaction.dealer

很抱歉我們實在無法理解題意,就我的認知,這是一個製造商的系統,為什麼會有經銷商的倉儲資訊,我不是很理解。
因此我的作法是,在vehicles表中設計一個transaction_date欄位,也就是汽車賣給經銷商的時間,經銷商拿到車之後,
可能會先存放,也可能直接賣,我把vehicles表與紀載交易的transaction表join起來,把經銷商拿到的時間減去賣給客戶的時間,
總體越高的可能就是存放最久的經銷商。

指令總整理:

1.SELECT vehicles.`VIN`,transaction.customer FROM `vehicles` INNER JOIN transaction ON vehicles.VIN=transaction.VIN WHERE `transmission` in (SELECT serial FROM transmissions WHERE `supplier` = '1')
2.SELECT dealer FROM transaction WHERE year(`date`)=year(now())-1 group by dealer ORDER by SUM(price) DESC LIMIT 0,1
3.SELECT brand FROM transaction WHERE year(`date`)=year(now())-1 group by brand ORDER by SUM(price) DESC LIMIT 0,2
4.SELECT month(date) FROM `transaction` WHERE `brand`='SUVs' GROUP BY month(`date`) ORDER BY SUM(price) DESC LIMIT 0,1
5.SELECT DATEDIFF(transaction.date, vehicles.transaction_date) AS DiffDate,transaction.dealer FROM transaction INNER JOIN vehicles ON transaction.VIN=vehicles.VIN GROUP BY transaction.dealer
